# プライバシーポリシー

本プライバシーポリシーは、「Cocotto」（以下、「本アプリ」といいます）の利用に際して、開発者が収集するユーザーの情報とその取り扱いについて説明するものです。本アプリをご利用いただく前に、本ポリシーをよくお読みいただき、内容にご同意いただく必要があります。

## 1. 収集する情報と利用目的

本アプリは、以下の目的のために、次の情報を収集することがあります。

### 1.1. ユーザーが自ら入力する情報
スポット（場所）の記録というコア機能を提供するために、ユーザーが直接入力する情報を収集します。
- **テキスト情報**: スポットの名称、感想・メモなどのコメント、評価（レーティング）、分類用のタグ
- **写真データ**: ユーザーがスポットに関連付けて、自身のデバイスからアップロードした写真

### 1.2. デバイスから自動的に取得する情報
アプリの機能を提供するために、デバイスから以下の情報を自動的に取得します。
- **位置情報（緯度・経度）**:
    - **フォアグラウンドでの利用**:
        - **目的**: 地図上にユーザーの現在地を表示するため。新しいスポットを現在地で登録するため。
    - **バックグラウンドでの利用**:
        - **目的**: ユーザーがジオフェンス機能を有効にした際に、特定のエリアへの出入りを検知し、通知を送信するため。
        - (バックグラウンドでの位置情報取得は、ユーザーがOSの設定で明示的に許可した場合にのみ実行されます。)

### 1.3. サービスの利用状況に関する情報
今後のサービス改善や機能追加に備え、以下の情報を収集する可能性があります。
- **匿名の利用状況データ**:
    - **目的**: サービスの改善、不具合の原因分析、利用動向の把握のため。
    - **内容**: クラッシュレポート、特定の機能の利用頻度などの統計情報。これらは個人を特定できない形で収集・利用されます。
- **広告識別子（IDFAなど）**:
    - **目的**: 将来的に広告を掲載した場合に、ユーザーの興味関心に合わせた広告（ターゲティング広告）を表示するため。

## 2. 情報の第三者提供

当開発者は、以下の場合を除き、ユーザーの個人情報を第三者に提供することはありません。

*   ユーザーの同意がある場合。
*   法令に基づき開示が求められた場合。
*   本アプリは、地図表示および場所検索機能を提供するために、Google LLCが提供するGoogle Maps/Places APIを利用しています。これに伴い、Googleがユーザーの情報を同社のプライバシーポリシーに基づいて収集する場合があります。詳細は、Googleのプライバシーポリシーをご確認ください。
    *   [Google プライバシーポリシー](https://policies.google.com/privacy)

## 3. ユーザーの権利

ユーザーは、本アプリ内で記録したご自身のデータ（スポット、コメント、写真など）をいつでも確認、修正、削除することができます。データの削除は、アプリ内の各詳細画面または一覧画面から行えます。

また、デバイスのOS設定から、本アプリの位置情報へのアクセス許可をいつでも変更・無効にすることができます。ただし、その場合、位置情報を利用する一部の機能が制限されることがあります。

## 4. 情報の管理

収集した情報の紛失、破壊、改ざん、および漏洩などを防止するため、不正アクセス対策、コンピュータウイルス対策など、適切な情報セキュリティ対策を講じます。

## 5. プライバシーポリシーの変更

開発者は、法令の変更やサービスの改善に伴い、本プライバシーポリシーを改定することがあります。重要な変更がある場合は、アプリ内での通知など、分かりやすい方法でお知らせします。
